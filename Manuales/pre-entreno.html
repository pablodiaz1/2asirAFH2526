Pre-entrenamiento con el Quijote

Manual de práctica: entrenar un modelo con texto del Quijote para que aprenda su estilo de escritura.

Práctica IA · Pre-entreno Windows · Python · PyTorch Autor: Pablo Diaz Castellanos

¿Qué es el pre-entrenamiento?

El pre-entrenamiento consiste en entrenar un modelo con nuevos datos para que aprenda un estilo o conocimiento específico. En esta práctica se entrena con texto del Quijote para que aprenda su estilo de escritura.

Idea clave

Si el modelo “absorbe” el estilo del Quijote, al pedirle texto se notará el lenguaje y la forma de narrar.

1) Crear carpeta del proyecto

mkdir quijote_ia
cd quijote_ia

Creamos la carpeta y entramos dentro para empezar el proyecto.

2) Crear y activar entorno virtual

python -m venv .venv
.venv\Scripts\activate

Aísla dependencias del proyecto para evitar conflictos con otras prácticas.

3) Instalar PyTorch

pip install torch

Instalamos la librería necesaria para entrenar el modelo con el dataset.

4) Entrenar con el Quijote

notepad entrenar_quijote.py
python entrenar_quijote.py

Ejecutamos el entrenamiento. Aquí se aplica el pre-entreno al modelo.

5) Probar modelo y cambiar el prompt

notepad probar_modelo.py
python probar_modelo.py
Comprobación

Cambia el prompt varias veces y observa que el modelo responde de forma distinta pero con estilo “quijotesco”. Repite una vez más para confirmar.

Conclusión

El pre-entrenamiento permitió adaptar el modelo para generar texto con el estilo del Quijote. Esto demuestra que es posible personalizar modelos de lenguaje mediante entrenamiento con datos específicos, y especializar modelos de IA para tareas concretas o estilos específicos.
